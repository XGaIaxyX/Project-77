Project on Console Screen

Visual Studio Code is a app where you can code multiple apps and even check them out by installing the live server extension. This app is even more advanced than Thunkable and Code.org because now you have to type, not use drag and drop blocks!


The Console Screen is where you can store data in your app! To open console screen, right click and then click on inspect. Once you're done with that, click on the top where it says console. To have data pop up in your console screen, you need to use the code "console.log();"
